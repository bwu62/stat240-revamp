2 Setup

This page will take you through all steps necessary to fully setup your computer. There are several things we need to do:

  1. First, in section 2.1 we will install both R and Rstudio.
  2. Next, in section 2.2 we will configure Rstudio and install some necessary R packages.
  3. Finally, in section 2.3 we will setup an organization system for your files.

2.1 R/Rstudio setup

We will be using both R and Rstudio throughout the course.

  • R is a free and open-source statistical computing software.
  • Rstudio is an IDE (integrated development environment) that makes developing in R much easier.
    • If you prefer, you can also use a different IDE like Visual Studio or Jupyter.

Note: R and Rstudio are two different programs that both need to be installed!

The instructions below are separated by operating system, depending on if you have a Windows or Mac machine.

  • If you have a Linux system, follow one of the linked instructions here.
  • If you have a Chromebook, try these steps.
